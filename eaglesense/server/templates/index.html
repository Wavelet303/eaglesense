EagleSense



EagleSense is a real-time top-view tracking system designed to be robust, accurate, and portable using off-the-shelf, non-invasive tracking infrastructures. It combines depth and infrared data for human posture and activity recognition. It provides a minimal RESTful API that can be used by proxemic-aware or cross-device systems to detect and localize users and their activities, as well as recognize their uses of devices within an interactive space.

Publications

EagleSense: Real-time Human Posture and Activity Recognition for Interactive Spaces using Top-View Depth-Sensing Cameras

Authors: Chi-Jui Wu, Steven Houben, Nicolai Marquardt @ University College London
